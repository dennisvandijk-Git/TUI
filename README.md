# 💻 TUI (Terminal User Interface)

A PowerShell-based TUI tool for daily administrative tasks.

## ✨ Features
- Interactive menu
- User onboarding
- System management

## ⚙️ Requirements
- PowerShell 5.1+
- UTF-8 with BOM (for special symbols)
